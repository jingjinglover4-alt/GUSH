# 📊 每日工作计划与反思系统

## 🎯 系统概述

这是一个**强迫思考、改变习惯、跟踪进度**的每日工作计划系统。系统通过以下方式帮助你：

1. **每日固定提醒** - 强迫你按时填写
2. **结构化反思** - 强迫你深度思考
3. **进度自动跟踪** - 强迫你对比完成情况
4. **数据持久化** - 强迫你形成习惯

## 🚀 快速开始

### 1. 立即使用
```bash
# 打开工作计划页面
python start_daily_plan.py
```

### 2. 设置每日自动提醒

#### macOS / Linux
```bash
# 运行自动化设置脚本
chmod +x setup_automation_mac_linux.sh
./setup_automation_mac_linux.sh
```

#### Windows
```powershell
# 以管理员身份运行PowerShell
# 然后执行：
.\setup_automation_windows.ps1
```

## 📝 核心功能

### 1. 今日工作内容
- **已完成的工作**：详细记录当天完成的任务
- **关键成果**：突出最重要的3个成果
- **遇到的挑战**：识别困难和障碍

### 2. 明日工作计划
- **计划工作**：按优先级排序的明日任务
- **目标设定**：遵循SMART原则（具体、可衡量、可实现、相关、有时限）
- **预见的风险**：提前识别风险并制定应对措施

### 3. 今日深度思考
- **收获与感悟**：今天学到的最重要一课
- **改进想法**：如果重新做，会有什么不同
- **感恩与认可**：培养感恩心态

### 4. 管理学视角
- **时间管理矩阵**：紧急重要矩阵分析
- **优先级矩阵**：Eisenhower矩阵评估
- **干系人管理**：沟通效果评估
- **决策质量**：数据驱动决策分析
- **团队协作**：合作亮点与改进点
- **领导力体现**：领导力行为识别

## 📊 进度跟踪系统

### 自动进度比较
系统自动对比前后两天数据：
1. **昨日计划完成度** - 计算完成百分比
2. **目标设定质量** - SMART原则符合度
3. **反思深度分析** - 思考质量评估
4. **连续打卡记录** - 习惯养成追踪

### 进度评分
- 🟢 **90%+** 优秀完成
- 🟡 **70%-89%** 良好完成  
- 🔴 **<70%** 需要改进

## 🛠️ 故障排除

### 常见问题

#### 1. Python脚本无法运行
```bash
# 检查Python安装
python --version

# 如果没有安装，从官网下载：
# https://www.python.org/downloads/
```

#### 2. 浏览器无法打开本地文件
- 确保文件路径没有中文或特殊字符
- 尝试使用Chrome或Edge浏览器
- 检查文件权限

#### 3. 计划任务不运行
```bash
# macOS/Linux 检查cron/launchd
launchctl list | grep dailyworkplan  # macOS
crontab -l                           # Linux

# Windows 检查任务计划程序
# 打开"任务计划程序"查看任务状态
```

#### 4. 数据丢失
- 系统每分钟自动备份
- 检查浏览器LocalStorage
- 导出重要数据定期备份
